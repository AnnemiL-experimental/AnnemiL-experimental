## About · 关于

I'm a biomedical engineering student drawn to the boundary between neuroscience and engineering — particularly how we might build interfaces for minds that work differently. Outside the lab, I collect mythologies, haunt humanities archives, and think about consciousness from too many angles at once.

生物医学工程学生。实验室里研究脑机接口与组织光学，实验室外收集神话体系、人文典籍，以及对意识问题想得太多。
